GitHub Flavored Markdown (GFM) Guide

GitHub Flavored Markdown (GFM) is an extension of the standard Markdown syntax, with additional features and functionalities tailored for the GitHub platform. It allows you to format text, create links, add images, and more.

## Basic Text Formatting

### Headers

# H1
## H2
### H3
#### H4
##### H5
###### H6

### Emphasis

*italic*
**bold**

### Lists

#### Unordered Lists

- Item 1
- Item 2
  - Subitem A
  - Subitem B

#### Ordered Lists

1. Item 1
2. Item 2
   1. Subitem A
   2. Subitem B

## Links and Images

### Links

[Link Text](https://www.example.com)

### Images

![Alt Text]()

### Relative Links and Images

[Link to File]()
![Image]()

## Code

### Inline Code

Wrap inline code with backticks:

Use `git clone` to clone a repository.

### Code Blocks

Indent code blocks with four spaces or use triple backticks:

    ```
    function example() {
        console.log("Hello, World!");
    }
    ```

Or

```javascript
function example() {
    console.log("Hello, World!");
}
